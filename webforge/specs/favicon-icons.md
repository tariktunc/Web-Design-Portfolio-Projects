# Favicon & Icon Generation Specification

> Tek kaynak gorsel → tum boyutlar otomatik uretilir.

---

## Required Icons

| File | Size | Usage |
|------|------|-------|
| `favicon.ico` | 32x32 | Browser tab (legacy) |
| `icon.svg` | scalable | Modern browsers (preferred) |
| `icon-192.png` | 192x192 | Android Chrome, PWA |
| `icon-512.png` | 512x512 | PWA splash screen |
| `apple-touch-icon.png` | 180x180 | iOS bookmark/home screen |
| `og-default.jpg` | 1200x630 | Open Graph share image |

## Next.js App Router — Auto Generation

```
src/app/
├── favicon.ico          → Browser tab icon
├── icon.tsx             → Dynamic icon generation (optional)
├── apple-icon.tsx       → Dynamic Apple icon (optional)
└── opengraph-image.tsx  → Dynamic OG image (optional)
```

### Option A: Static Files (Simplest)

Place files directly in `src/app/`:
```
src/app/favicon.ico          → auto-served at /favicon.ico
src/app/icon.png             → auto-served, auto <link> tag
src/app/apple-icon.png       → auto-served, auto <link> tag
```

Next.js automatically generates the correct `<link>` tags in `<head>`.

### Option B: Dynamic Generation (From SVG Source)

```tsx
// src/app/icon.tsx
import { ImageResponse } from 'next/og';

export const size = { width: 32, height: 32 };
export const contentType = 'image/png';

export default function Icon() {
  return new ImageResponse(
    <div style={{
      width: 32, height: 32,
      background: 'var(--primary, #2563EB)',
      borderRadius: 6,
      display: 'flex', alignItems: 'center', justifyContent: 'center',
      color: 'white', fontSize: 20, fontWeight: 'bold',
    }}>
      M
    </div>,
    { ...size }
  );
}
```

### Option C: CLI Generation (Build Time)

```bash
# Generate all icons from a single source SVG/PNG
# Using sharp (Node.js image processing)
npx tsx scripts/generate-icons.ts
```

```typescript
// scripts/generate-icons.ts
import sharp from 'sharp';
import { readFileSync } from 'fs';

const SOURCE = 'src/assets/logo-source.png'; // or .svg

async function generateIcons() {
  const source = readFileSync(SOURCE);

  // favicon.ico (32x32)
  await sharp(source).resize(32, 32).toFile('src/app/favicon.ico');

  // icon-192 (Android/PWA)
  await sharp(source).resize(192, 192).png().toFile('public/icon-192.png');

  // icon-512 (PWA splash)
  await sharp(source).resize(512, 512).png().toFile('public/icon-512.png');

  // apple-touch-icon (180x180)
  await sharp(source).resize(180, 180).png().toFile('src/app/apple-icon.png');

  console.log('All icons generated.');
}

generateIcons();
```

## Web Manifest (PWA)

```json
// public/manifest.json (or src/app/manifest.ts for dynamic)
{
  "name": "Site Adi",
  "short_name": "Site",
  "description": "Site aciklamasi",
  "start_url": "/",
  "display": "standalone",
  "background_color": "#ffffff",
  "theme_color": "#2563EB",
  "icons": [
    { "src": "/icon-192.png", "sizes": "192x192", "type": "image/png" },
    { "src": "/icon-512.png", "sizes": "512x512", "type": "image/png" }
  ]
}
```

### Dynamic Manifest (Next.js)
```typescript
// src/app/manifest.ts
import type { MetadataRoute } from 'next';

export default function manifest(): MetadataRoute.Manifest {
  return {
    name: 'Site Adi',
    short_name: 'Site',
    description: 'Site aciklamasi',
    start_url: '/',
    display: 'standalone',
    background_color: '#ffffff',
    theme_color: '#2563EB',
    icons: [
      { src: '/icon-192.png', sizes: '192x192', type: 'image/png' },
      { src: '/icon-512.png', sizes: '512x512', type: 'image/png' },
    ],
  };
}
```

## Head Tags (Auto-Generated by Next.js)

When files are placed correctly, Next.js generates:
```html
<head>
  <link rel="icon" href="/favicon.ico" sizes="32x32" />
  <link rel="icon" href="/icon.png" type="image/png" sizes="32x32" />
  <link rel="apple-touch-icon" href="/apple-icon.png" sizes="180x180" />
  <link rel="manifest" href="/manifest.json" />
  <meta name="theme-color" content="#2563EB" />
</head>
```
