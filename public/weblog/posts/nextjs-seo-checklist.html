Next.js SEO Checklist: Proje Başlangıç Kontrol Listesi

Next.js, SEO dostu web uygulamaları geliştirmek için mükemmel bir framework'tür. Sunucu tarafı rendering, statik site oluşturma ve dahili optimizasyon araçları sayesinde arama motorlarında üst sıralara çıkmanızı kolaylaştırır. Bu rehberde yeni bir Next.js projesi başlatırken uygulamanız gereken SEO kontrol listesini adım adım ele alacağız.

// app/blog/[slug]/page.js
export async function generateMetadata({ params }) {
  const post = await getPost(params.slug);

  return {
    title: post.title,
    description: post.excerpt,
    openGraph: {
      title: post.title,
      description: post.excerpt,
      type: 'article',
      publishedTime: post.publishedAt,
      modifiedTime: post.updatedAt,
      authors: [post.author.name],
      images: [{ url: post.coverImage, width: 1200, height: 630 }],
    },
    alternates: {
      canonical: `/blog/${params.slug}`,
    },
  };
}

2. Sitemap ve Robots.txt

// app/sitemap.js — Dinamik sitemap
export default async function sitemap() {
  const posts = await getAllPosts();
  const pages = await getAllPages();

  const postUrls = posts.map(post => ({
    url: `https://www.example.com/blog/${post.slug}`,
    lastModified: new Date(post.updatedAt),
    changeFrequency: 'weekly',
    priority: 0.8,
  }));

  const pageUrls = pages.map(page => ({
    url: `https://www.example.com/${page.slug}`,
    lastModified: new Date(page.updatedAt),
    changeFrequency: 'monthly',
    priority: 0.6,
  }));

  return [
    {
      url: 'https://www.example.com',
      lastModified: new Date(),
      changeFrequency: 'daily',
      priority: 1.0,
    },
    ...postUrls,
    ...pageUrls,
  ];
}
// app/robots.js — Dinamik robots.txt
export default function robots() {
  return {
    rules: [
      {
        userAgent: '*',
        allow: '/',
        disallow: ['/api/', '/admin/', '/private/'],
      },
    ],
    sitemap: 'https://www.example.com/sitemap.xml',
  };
}

3. Structured Data (JSON-LD)

// components/JsonLd.js
export default function JsonLd({ data }) {
  return (
    <script
      type="application/ld+json"
      dangerouslySetInnerHTML={{ __html: JSON.stringify(data) }}
    />
  );
}

5. URL Yapısı ve Yönlendirmeler

// next.config.js — Redirects ve rewrites
module.exports = {
  async redirects() {
    return [
      // www yönlendirmesi (Vercel'de otomatik)
      {
        source: '/:path*',
        has: [{ type: 'host', value: 'example.com' }],
        destination: 'https://www.example.com/:path*',
        permanent: true, // 301
      },
      // Eski URL'lerden yeni URL'lere
      {
        source: '/blog/old-post-url',
        destination: '/blog/new-post-url',
        permanent: true,
      },
      // Trailing slash tutarlılığı
      {
        source: '/blog/:slug/',
        destination: '/blog/:slug',
        permanent: true,
      },
    ];
  },

  // trailing slash yapılandırması
  trailingSlash: false,
};

İpucu: Her sayfada yalnızca bir adet <h1> kullanın. Heading hiyerarşisini (h1 > h2 > h3) doğru sıralayın. Bu hem erişilebilirlik hem de SEO için kritiktir.

8. SEO Kontrol Listesi Özeti

  • Metadata: Her sayfada benzersiz title ve description
  • Canonical URL: Her sayfada canonical tag
  • Sitemap: Dinamik sitemap.xml
  • Robots.txt: Doğru yapılandırma
  • Structured Data: JSON-LD schema markup
  • Open Graph: Sosyal medya kartları
  • Resim Optimizasyonu: Alt text, format, boyut
  • Core Web Vitals: LCP, INP, CLS hedefleri
  • HTTPS: Güvenli bağlantı
  • Mobil Uyumluluk: Responsive tasarım
  • URL Yapısı: Temiz, açıklayıcı URL'ler
  • Heading Hiyerarşisi: Tek h1, doğru sıralama
  • İç Linkleme: Anlamlı anchor text'ler
  • 404 Sayfası: Özel not-found sayfası
  • Yönlendirmeler: 301 redirect'ler
Uyarı: noindex meta tag'ını production'a deploy etmeden önce kaldırdığınızdan emin olun. Staging ortamlarında kullanılan robots: noindex production'da büyük SEO hasarına yol açar.

Sonuç

Next.js, SEO dostu uygulamalar geliştirmek için kapsamlı araçlar sunar. Metadata API ile dinamik meta tag'lar, sitemap ve robots.txt oluşturma, Image component ile resim optimizasyonu ve ISR ile güncel içerik sunma bu araçların başlıcalarıdır. Bu kontrol listesini projenizin başlangıcında uygulayarak SEO temellerini sağlam bir şekilde atabilirsiniz. Düzenli olarak Google Search Console ve PageSpeed Insights ile metriklerinizi izleyin.
